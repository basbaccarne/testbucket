## General
Een HC05 module is een bluetooth module die rechtstreeks kan verbonden worden met UART pinnen (Rx/Tx). Op die manier kan je draadloos communiceren op een manier die erg lijkt op de seriele communicatie via een USB verbinding.

## Satus light interpretatie
* Rapid flash: powered a trying to establish connection
* Slow blink: command mode (see below)
* Periodic short burst: connected

## Een HC05 module in AT command mode zetten
Wanneer je de instellingen van je HC05 wil bekijken of aanpassen moet je deze in command mode zetten en een communicatie via een serial to USB interface (e.g. [CP2102](https://www.otronic.nl/nl/cp2102-usb-naar-ttl-serial-uart-bridge-33v-5v.html))) opzetten.   

**Set-up:**
* Gebruik een serial to USB converter
* Verbind Vcc, Ground, RX(D) & TX(D)
* Default settings:
    * Baud Rate 38300
    * 8 Data bits
    * Parity: none
    * Stop bits: 1
    * CR + LF
* Boot de module in AT mode door het knopje in te drukken bij het maken van de USB verbinding
* In terminal (Aruino IDE, VSC, Coolterm, Putty, ...): test door de boodschap `AT` te zenden. Je hoort `OK` te krigen als antwoord ✅

**AT commands:**
* `AT+UART=9600,0,0` set baud rate to 9600 (baud = 9600, stop bits = 1, parity = none)
* `AT+ROLE=0` Set to slave mode: Slave mode is normal for Arduino → phone or PC connects to it (1 = master)
* `AT+CMODE=1`Allow any master to connect (1 = any master, 0 = specific master only)
* `AT+NAME=MyHC05` Set name of the module
* `AT+ORGL`reset everything back to facotry settings
* `AT+PSWD=1234` set password (1234 is the default)
* `AT+RESET` restart and go to data mode (exit configurator)

You can also check the current setting with a questionmark (e.g. `AT+NAME?`)
